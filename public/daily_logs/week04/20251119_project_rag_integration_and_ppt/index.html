4. 개인 인사이트 (Reflection)#

  • 배운 점: RAG 구현 시 LangChain 라이브러리 전체를 가져오지 않고도, ChromaDB와 OpenAI API만으로 핵심적인 검색-생성 구조를 독립적으로 구축할 수 있음을 확인했습니다.
  • 느낀 점: Streamlit에서 llm_utils나 vector_db를 @st.cache_resource로 관리하면, LLM 호출 전에 DB를 매번 초기화할 필요가 없어 성능이 대폭 향상될 것으로 기대됩니다.
